PokerGame is a terminal-based multiplayer Poker game supporting up to five players. The main objetive of this project was to implement a robust network synchronization and asynchronous communication using the Asio Library.

The core of this project is its networking layer, designed to handle real-time game state distribution and client-server interaction in a terminal environment.

The network library was built following a architecture inspired by the design patters proposed by Javidx9 (OneLoneCoder). Available at: https://github.com/OneLoneCoder/Javidx9/tree/master/PixelGameEngine/BiggerProjects/Networking)

For the hand evaluator the choosen algorithm was the OMPEval. Available at: https://github.com/zekyll/OMPEval
The OMPEval repository was cloned into the folder "evaluator" and was modified in a way that only the needed files were present. To more information, visit the original repository at https://github.com/zekyll/OMPEval
